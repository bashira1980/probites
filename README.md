## 💻 Local Preview

To view this website locally on your computer:

### Option 1: Double-Click (Simplest)
1. Download or clone this repository
2. Navigate to the folder
3. Double-click `index.html`
4. The website will open in your default browser

### Option 2: Using a Local Server (Recommended)
For the best experience with all features working correctly:

**Using Python (Mac/Linux):**
```bash
# Navigate to the project folder
cd probite-website

# Python 3
python3 -m http.server 8000

# Then open browser to: http://localhost:8000
```

**Using Node.js:**
```bash
# Install live-server globally (one time)
npm install -g live-server

# Run from project folder
live-server
```

**Using VS Code:**
- Install "Live Server" extension
- Right-click `index.html`
- Select "Open with Live Server"
